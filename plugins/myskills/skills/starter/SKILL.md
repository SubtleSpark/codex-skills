# Starter

Use this skill only when the task is to confirm that the `myskills` plugin is installed or to briefly explain this repository's structure.

When invoked:

1. State that `myskills` is installed as a plugin.
2. Mention that bundled skills live under `plugins/myskills/skills/`.
3. Keep the response short and factual.
